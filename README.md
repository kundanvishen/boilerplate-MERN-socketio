
## A very-handy boilerplate to get started with a production level react and node projects

With following configuration:

- Node
- ES6
- ESLint
- Flow
- Jest
- Husky
- Express
- Nodemon
- PM2
- Webpack
- React
- HMR
- Redux
- Immutable
- Fetch
- SSR
- Helmet
- SocketIO
- Bootstrap
- Travis
- Coveralls
- Heroku

Author: [Kundan Singh](https://github.com/kundanvishen)
